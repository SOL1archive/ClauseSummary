
# Parser
## Parser pseudo code
1. 목차 있음 -> 각 단위로 분리(spilt)
    1. 각 목차로 분리한 것들에 대해
        1. 목차 있음 -> 해당 페이지 지우기
        2. 만약 다음 페이지가 '제1조'로 시작 x -> 해당 페이지 날리기
        3. 2번 반복
    2. '제1조' 에서 시작
    3. '(마지막)조'에서 끝
    4. 추출
2. 각 단위 concat

## 목차 분리시 자료 구조
`[목차 단위][페이지]`

## Assumed Text Format
if '목' in
<보통>
제1조 ...
...

<특별>
제1조 ..
...

제 마지막 조
